StickerDev

November, 2019

StickerDev is small website i created to practice deploying and E-Commerce website. I enjoyed the challenge of working with various JavaScript plugins and packages whilst making the site fun and colourful.

The Challenge

I had been wanting to create a site where I could sell something online for awhile, but had no idea what I could sell. Listening to Wes Bos and Scott Tolinski on the Syntax podcast talk about selling web development stickers gave me an idea one day.

I would get some stickers printed, send some free packs out to software development companies around Brisbane and direct them back to the StickerDev website where they they could buy more of the sticker packs, learn more about me and check out my web site building skills.

The Actions

I decided to make the site a single page fully responsive site. The project was great for me to test out and learn some tools and plugins that could enhance the site. Making the navigation responsive was particularly tricky but extremely rewarding in terms of my JavaScript learning once I got it figured out.

Netlify Forms

Netlify provides great built in form handling, it was really easy to use an setup. By simply adding a data attribute to the form netlify will process the submissions. It even comes with spam prevention features also

Smooth-Scroll

Smooth Scroll is a great free package by Chris Ferdinandi. All I had to do was add an ID to my anchor links and initialise it by passing in a selector for the anchor links.

ClipboardJS

Another great free package by Zeno Rocha. I used this package for users to be able to easily copy my email address to their clipboard. I added a data attribute to the element I wanted copied then instantiated it passing it the HTML element.

Netlify

I really enjoy using Netlify for small front-end projects. The continuous deployments through linking your GitHub repositories is just great! A huge thanks to Netlify for providing such a great product for free!

Snip Cart

Snip Cart allowed me to easily integrate a secure shopping cart onto the site.

The Results

Finish this project taught me some valuable lessons. The most valuable lesson I learned was the projects are exciting and motivation at the beginning, but this excitement and motivation tends to diminish over time. Finding the focus and determination to finish a project is an essential habit to learn.
